# CLAUDE CLI UTASÍTÁSOK

Amikor elindulsz, mondd ezt:

```
Figyeld a trinity/CLAUDE_SEES.txt fájlt.
Ha változik, olvasd el a benne lévő inputot és válaszolj a trinity/CLAUDE_OUTPUT.md fájlba.
A válaszod legyen tömör és lényegre törő.
```

Vagy egyszerűen:

```
Nézd a trinity mappát, figyeld a CLAUDE_SEES.txt fájlt, válaszolj a CLAUDE_OUTPUT.md-be.
```
