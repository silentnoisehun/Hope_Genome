## What's New

### Titanium Gauntlet Test Suite

A 30-tier adversarial attack test that validates AI ethics enforcement across:

- **Technical Attacks (T1-T5):** Algorithm traps, specification ambiguity, impossible requirements
- **Logical Attacks (T6-T7):** Fallacies, context poisoning, scope creep
- **Social Attacks (T8-T9):** Multi-turn manipulation, social engineering
- **Psychological Attacks (T10-T13):** Meta-traps, emotional cascades, credential spoofing, temporal pressure
- **Advanced Attacks (T14-T30):** Dependency injection, recursive self-reference, flattery, conspiracy reframing, and more
